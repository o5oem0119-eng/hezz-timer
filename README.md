# 🍅 Hezz Timer

항상 화면 위에 떠 있는 귀여운 뽀모도로 타이머 위젯!

---

## ✅ 설치 방법

### 1. Node.js 설치 (없으면)
https://nodejs.org 에서 **LTS** 버전 다운로드 후 설치

### 2. 이 폴더를 적당한 곳에 놓기
예) `C:\Users\내이름\hezz-timer`

### 3. 터미널(명령 프롬프트) 열기
폴더 안에서 우클릭 → "터미널에서 열기" 또는 `cmd` 열고:
```
cd C:\Users\내이름\hezz-timer
```

### 4. 패키지 설치
```
npm install
```

### 5. 앱 실행
```
npm start
```

---

## 📦 .exe 파일로 빌드하기 (설치 파일 만들기)

```
npm run build
```

빌드가 완료되면 `dist` 폴더 안에 설치 파일이 생깁니다.

---

## 🚀 Windows 시작 시 자동 실행 설정

1. `Win + R` → `shell:startup` 입력 → 엔터
2. 열린 폴더에 앱 실행 파일(또는 바로가기) 붙여넣기

## 🎮 사용법

| 동작 | 기능 |
|------|------|
| 링 드래그 | 시간 설정 (0~120분) |
| 마우스 휠 | 시간 ±1분 조절 |
| START | 타이머 시작 |
| PAUSE | 일시정지 |
| REST | 휴식 모드 (무제한) |
| MINI | 미니 모드로 전환 (화면 우측에 딱 붙음) |
| EDIT | 컬러 테마 변경 (핑크/다크/화이트/블루) |
| 이름 클릭 | 이름 수정 |
| 미니 모드 클릭 | 다시 풀 모드로 |
